# 個人房貸試算器

透過使用者輸入房屋總價、自備款、貸款利率、貸款年限及寬限期，進行精確的本息平均攤還計算，提供使用者一目了然的還款分析報告。


## 核心功能

### 1. 貸款相關資訊計算
* **貸款總額分析**：自動計算扣除自備款後的實際貸款金額。
* **還款結構拆解**：精確計算首期應繳之「利息」與「本金」，讓使用者掌握初期還款負擔。
* **長期成本試算**：統計全期總利息支出與最終總還款金額（含本金與利息）。

### 2. 支援寬限期
* **動態兩階段試算**：參考市面常見方案，提供 0~10 年寬限期設定。
* **智能提示**：當設定寬限期時，結果區下方會顯示小提醒，說明寬限期內僅需負擔利息，並標註期滿後恢復本息攤還的數值。

### 3. 防呆機制
* **資料格式驗證**：資料格式驗證：利用 `decimal.TryParse` 有效過濾非數字、負數或空值等不合理輸入並彈出警告視窗提醒使用者。
* **邏輯錯誤攔截**：當自備款超過總價，或寬限期大於等於貸款年限時，系統會彈出警告視窗防止錯誤計算。

### 4. 無滑鼠操作
* **開啟即輸入**：啟動後自動聚焦於「房屋總價」欄位，手不用離開鍵盤即可開始打字。
* **流暢切換**：全程支援 `Tab` 鍵跳轉、方向鍵選取自備款、空白鍵確定按鈕以及 `Enter` 鍵執行與關閉視窗。


## 使用說明

1.  **資料輸入**：
    * **房屋總價(萬元)**：預設為 1000，必須為大於 0 的正數。
    * **自備款模式(二選一)**：預設為「比例 20%」，可透過 **鍵盤左鍵** 切換模式。
      * **比例(%)**：輸入範圍為 0~100。
      * **金額(萬元)**：輸入範圍必須為正數且小於房屋總價。
    * **貸款利率(%)**：預設為 2.15，參考市面貸款利率將範圍設成 0~20。
    * **貸款年限(年)**：預設為 30，可手動輸入或透過 **鍵盤上下鍵/點擊箭頭** 修改數值，參考市面實際貸款年限將範圍設成 0~50。
    * **寬限期(年)**：預設為 0，可手動輸入或透過 **鍵盤上下鍵/點擊箭頭** 修改數值，參考市面實際寬限期將範圍設成 0~10，且必須小於貸款年限。

## 公式參考

本程式採用標準「本息平均攤還法」公式進行計算：

| 項目 | 公式 |
| :--- | :--- |
| **月利率 (r)** | 年利率 / 12 |
| **總月數 (n)** | (貸款年限 - 寬限期) / 12 |
| **貸款總金額 (P)** | 房屋總價 - 自備款 |
| **首期本金** | 若有寬限期為 $0$，無寬限期則為 每月應繳額 - 首期利息 |
| **首期利息** | $P \times r$ |
| **每月應繳額** | $P \times \frac{r \times (1+r)^n}{(1+r)^n - 1}$ |
| **總利息支出** | $(\text{首期利息} \times \text{寬限期月數}) + (\text{每月應繳額} \times n) - P$ |
| **總還款金額** | $P + 總利息支出$ |
